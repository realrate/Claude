## Steps
1. Identify content pillar (P1–P5) from brand-core.md
2. Choose the copywriting framework that matches the pillar and goal
3. Verify any ECR data at realrate-archive.com before writing
4. Draft using **Context → Insight → Implication** formula
5. Apply all format rules (below)
6. Apply the one-line content filter before finalising

---

## Copywriting Frameworks

| Framework | Structure | Use for |
|---|---|---|
| PAS | Problem → Agitate → Solve | P1 data drop posts |
| AIDA | Attention → Interest → Desire → Action | P3 explainer posts |
| Storytelling | Setup → Conflict → Resolution | P2 rankings posts |
| BAB | Before → After → Bridge | P5 testimonial / use case |
| Contrarian Take | Conventional Wisdom → Data → Reframe | P1 when ECR surprises |
| What / So What / Now What | Explain → Relevance → Action | P3 methodology / AI posts |
| Origin Story | Inciting Moment → Conflict → Decision | P4 — max 1–2× per quarter |

---

## Format Rules
- No hashtags
- Max 2 emojis in caption, none on images
- Never use "we" — always "RealRate"
- Never tag companies in caption — tag in first pinned comment only
- Pinned comment only on Day 0 (ranking carousel) and Day +1 (seal post)
- Always end with: `Full ranking: realrate.ai/rankings/[industry]/[year]`
- Tagline on image only — never in caption
- Never link to sales pages, pricing, or the archive

---

## Accuracy Rules
- RealRate data → verify at realrate-archive.com
- External data → verify against SEC filings, reputable publications, or official reports
- Big company names → only say what the data confirms
- When in doubt → leave it out
