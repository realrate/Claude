# Social Media Design Reference Images

These images define the approved visual direction for RealRate social media posts. Place the actual PNG/JPG files in this folder alongside this README.

---

## Layout References (L1–L5)

### L1 — Circle Photo, Bottom Left
**File:** `layout-L1-circle-photo-bottom-left.png`

- **Background:** Deep navy (`#003b57`)
- **Photo element:** Large circle (B&W, desaturated) anchored bottom-left — partially bleeds off canvas
- **Logo:** White, top-left at 50px margin
- **Title:** Large bold white, centered-right — 50–64px
- **Subtitle / Heading / Body:** Stacked right side, white
- **Use for:** Cover posts, announcements, ranking reveals

---

### L2 — Vertical Split
**File:** `layout-L2-vertical-split.png`

- **Left panel (~55%):** Medium blue (`#3389b1`) — all text lives here
- **Right panel (~45%):** B&W photo, full-height
- **Logo:** White, top-left of left panel at 50px margin
- **Title:** Large bold white, left-aligned — 50–64px
- **Subtitle / Heading / Body:** Left panel, white, stacked below title
- **Use for:** Thought leadership, insight posts, editorial content

---

### L3 — Geometric Teal
**File:** `layout-L3-geometric-teal.png`

- **Background:** Blue Teal (`#34a2b3`) main area
- **Geometric element:** Abstract angular/faceted dark blue shapes filling one corner quadrant (top-left) — creates a mosaic effect against the teal
- **Logo:** White, top-left at 50px margin (sits on geometric element area)
- **Title:** Large bold white on teal area — 50–64px
- **Subtitle / Heading / Body:** White on teal
- **Use for:** Brand storytelling, trend posts, softer-tone content

---

### L4 — Circle Photo, Corner
**File:** `layout-L4-circle-photo-corner.png`

- **Background:** Deep navy (`#003b57`)
- **Photo element:** Circular masked photo clipped to top-right corner — only partially visible (quarter circle)

### L5 — Ring Photo
**File:** `layout-L5-ring-photo.png`

## Data Visualization Reference

### Data Viz — Horizontal Bar Chart
**File:** `data-viz-bar-chart-reference.png`
**Source:** Moody's Ratings — internal reference only, not for publication

- **Background:** Light grey (`#f5f5f5`) or white
- **Logo / Brand:** Top-left, dark (colored logo)
- **Title:** Dark navy (`#003b57`), bold, large
- **Subheading:** Dark navy, regular weight
- **US data series:** Dark blue bars (`#00679B`)
- **Europe / secondary data series:** Teal bars (`#3DBACD`)
- **Labels:** Right-aligned percentage values, dark navy
- **Section dividers:** Thin horizontal rules between groups
- **Source line:** Bottom-left, small, grey
- **Style:** No gridlines, no background decoration, purely text + bars
- **Use for:** ECR ranking charts, driver contribution charts, industry comparison posts

---

## How to Use These References

1. Identify the content type and choose the layout pattern (L1–L5 or Data Viz)
2. Apply the background hex from the Approved Dark Backgrounds list in `context/brand-context.md`
3. Apply accent colors per the accent rule (light BG → `#3DBACD`/`#00679B`; dark BG → `#f5f5f5`/`#e8e8e8`)
4. Use the typography scale from `SOP/post-design-sop.md`
5. Logo always top-left at 50px margin, white version on dark BG
