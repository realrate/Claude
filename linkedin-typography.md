# RealRate LinkedIn Post Typography Rules

Apply to every RealRate LinkedIn post graphic (1080×1350px). No exceptions.

## Font Size Scale

| Element | Size | Notes |
|---|---|---|
| Headline | max 75px | |
| Body text — light slides (2–6) | 35px | Intro text, bullet text, signal/conclusion text |
| Body text — dark cover slide | 42px | Subhead and step preview text on cover |
| Subheadlines / questions | 28–32px | Italic where appropriate |
| Footer (brand name, tagline, URL) | 20–25px | |

## Layout Rules

- **No eyebrow labels / mini titles.** Never add any small label above or near the headline — no category tags, topic labels, overlines, or descriptors. Go straight into the heading.
- **Grey text → use white or strong dark.** On dark slides: all text must be full white (`#ffffff`). On light slides: body text must be strong dark (`#12293a`).
- **Cover content anchors at the top.** Content starts directly below the logo — do not center-push it to the middle.
- **Layout:** Use `justify-content: space-between` so sections are distributed evenly and fill the full 1350px height.
